# PolarSwarm

PolarSwarm is a local IssueOps orchestrator for multi-agent software development.

Current development mode: Vibe Coding + Spec-Guided Bootstrap.

## Documentation

Versioned documentation lives in [`docs/`](docs/README.md).

## Local Commands

```sh
go test ./...
go run ./cmd/polarswarm help
go run ./cmd/polarswarm version
```
